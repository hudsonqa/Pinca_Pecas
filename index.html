Bem-vindo ao Pinça Peças v2.2!

Sua ferramenta multifuncional para extrair e dividir documentos PDF com precisão e velocidade.

Agora, além da extração inteligente por peças, você pode dividir qualquer PDF em partes menores. Este manual cobre todas as funcionalidades.

Baixar Pinça Peças


Instalação e Primeiro Uso

Siga estes passos simples para começar a usar o Pinça Peças.

  1. 1. Baixe o Programa
    Clique no botão de download acima para baixar o arquivo Pinca_Pecas.zip.
  2. 2. Descompacte o Arquivo
    Localize o arquivo .zip em sua pasta de downloads. Clique com o botão direito sobre ele e escolha a opção "Extrair Tudo..." ou "Descompactar aqui".
  3. 3. Execute o Pinça Peças
    Abra a pasta que foi criada. Dentro dela, dê um duplo clique no arquivo Pinca_Pecas.exe para iniciar o programa.
Aviso na Primeira Execução

Como o Pinça Peças é um programa novo, o Windows pode exibir uma tela azul de proteção (SmartScreen) ao executá-lo pela primeira vez. Isso é normal.

Para prosseguir, clique em "Mais informações" e, em seguida, no botão "Executar assim mesmo". O programa é seguro e isso só precisa ser feito uma vez.

Dica de Ouro!

Para facilitar o acesso, crie um atalho para o Pinca_Pecas.exe em sua Área de Trabalho.


Navegando pela Interface

O Pinça Peças agora é organizado em abas. Clique em uma aba para alternar entre as funcionalidades.

Aba: Extração por Peças

Use esta aba para extrair documentos específicos (peças processuais) de um PDF, baseando-se nos marcadores (bookmarks) do arquivo. Ideal para processos bem estruturados.

Aba: Divisão por Páginas

Use esta aba para dividir qualquer PDF em arquivos menores com um número fixo de páginas. Perfeito para documentos longos ou sem marcadores.


Funcionalidades Detalhadas

Aba: Extração por Peças

Esta função permite extrair documentos com base em filtros de texto aplicados aos marcadores do PDF.

  1. Abra um PDF e veja a lista de marcadores em "Resultados".
  2. Adicione Filtros (ex: "Sentença") para refinar a lista.
  3. Selecione as peças que deseja extrair (use Ctrl+Clique).
  4. Escolha se quer salvar em arquivo único ou separadamente.
  5. Clique em "Executar e Salvar" ou "Processar Pasta..." para automação em lote.
Você também pode Salvar e Carregar seus conjuntos de filtros para reutilizá-los.
Aba: Divisão por Páginas

Esta função divide um PDF em vários arquivos menores, cada um com um número máximo de páginas definido por você.

  1. Clique em "Abrir PDF" e selecione o arquivo que deseja dividir.
  2. No campo "Dividir a cada", defina o número de páginas para cada novo arquivo (ex: 100).
  3. O programa informará quantos arquivos serão criados.
  4. Clique em "Dividir e Salvar...".
  5. Escolha a pasta onde os arquivos divididos serão salvos.


Dicas e Solução de Problemas

Use "Extração por Peças" se o seu PDF é um processo judicial com marcadores para cada documento (Petição, Contestação, etc.). Use "Divisão por Páginas" para qualquer outro PDF longo que você precise quebrar em pedaços menores, ou para processos que não tenham marcadores.

Isso significa que o PDF aberto não possui marcadores (bookmarks). Para este tipo de arquivo, a função de extração por peças não funcionará. Utilize a aba "Divisão por Páginas" para trabalhar com este documento.

A barra de progresso na parte inferior da janela deve indicar que o programa está trabalhando. Se ela não se mover por muito tempo, você pode usar o botão "Cancelar" para interromper a tarefa com segurança. Processar pastas com centenas de arquivos ou dividir um PDF muito grande pode levar algum tempo.
