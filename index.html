Bem-vindo ao Pinça Peças!

Sua ferramenta definitiva para extrair, organizar e otimizar documentos de processos em PDF com precisão e velocidade.

Este manual foi criado para guiá-lo por todas as funcionalidades do software. Comece baixando o programa no botão abaixo.

Baixar Pinça Peças Agora


Instalação e Primeiro Uso

Siga estes passos simples para começar a usar o Pinça Peças.

  1. 1. Baixe o Programa
    Clique no botão de download acima para baixar o arquivo Pinca_Pecas.zip.
  2. 2. Descompacte o Arquivo
    Localize o arquivo Pinca_Pecas.zip em sua pasta de downloads. Clique com o botão direito sobre ele e escolha a opção "Extrair Tudo..." ou "Descompactar aqui". Isso criará uma nova pasta com os arquivos do programa.
  3. 3. Execute o Pinça Peças
    Abra a pasta que foi criada no passo anterior. Dentro dela, encontre e dê um duplo clique no arquivo Pinca_Pecas.exe para iniciar o programa.
Aviso na Primeira Execução

Como o Pinça Peças é um programa novo, o Windows pode exibir uma tela azul de proteção (SmartScreen) ao executá-lo pela primeira vez. Isso é normal.

Para prosseguir, clique em "Mais informações" e, em seguida, no botão "Executar assim mesmo". O programa é seguro e isso só precisa ser feito uma vez.

Dica de Ouro!

Para facilitar o acesso, clique com o botão direito no arquivo Pinca_Pecas.exe, vá em "Enviar para" e selecione "Área de Trabalho (criar atalho)". Assim, você poderá iniciar o programa diretamente do seu desktop.


Conhecendo a Interface

A interface foi projetada para ser intuitiva. Ela é dividida em 4 áreas principais:

1. Arquivo

Área para abrir um único arquivo PDF e visualizar todos os seus marcadores.

2. Filtros

O cérebro da operação: aqui você adiciona, salva e carrega os termos para encontrar as peças desejadas.

3. Resultados

Exibe os documentos encontrados no PDF aberto, em sua ordem original, com o respectivo intervalo de páginas.

4. Ações e Salvamento

Onde a mágica acontece! Escolha como salvar e execute a extração para um arquivo ou para uma pasta inteira.


Funcionalidades Detalhadas

Uso de Filtros

Os filtros são o coração do Pinça Peças. Eles funcionam com uma lógica "OU": se você adicionar os filtros Petição e Sentença, o programa listará todos os documentos que contenham "Petição" OU "Sentença" em seus marcadores.

  • Adicionar Filtro: Digite o termo e pressione "Enter".
  • Remover Filtro: Clique no "x" do "badge" correspondente.
  • Ignorar Maiúsculas/Minúsculas: Marque esta caixa para que a busca por "petição" também encontre "Petição" e "PETIÇÃO". Por padrão, a busca diferencia as letras.
Salvar e Carregar Filtros

Economize tempo e mantenha a consistência em seu trabalho salvando seus conjuntos de filtros mais usados.

  • Salvar Filtros: Com os filtros desejados na tela, clique em "Salvar Filtros" e escolha um nome e local para seu arquivo .json.
  • Carregar Filtros: Clique em "Carregar Filtros", selecione um arquivo .json salvo anteriormente e seus filtros aparecerão instantaneamente.
Extração de Arquivo Único

Ideal para trabalhar com um processo específico que você tenha aberto.

  1. Clique em "Abrir PDF" e selecione o arquivo.
  2. Use os filtros para encontrar as peças desejadas na lista de resultados.
  3. Selecione as peças: Você pode clicar para selecionar uma ou mais peças na lista (use Ctrl+Clique ou Shift+Clique). Se nada for selecionado, o programa assume que você quer extrair TUDO da lista.
  4. Escolha a opção de salvamento ("arquivo único" ou "separadamente").
  5. Clique em "Executar e Salvar".
Dica: Os PDFs extraídos manterão os marcadores originais, facilitando a navegação!
Processamento em Lote (Pastas)

A funcionalidade mais poderosa para automação. Aplique seus filtros a centenas de PDFs de uma só vez.

  1. Importante: Você NÃO precisa abrir um PDF individual para esta função.
  2. Adicione ou carregue os filtros que deseja aplicar.
  3. Escolha a opção de salvamento. Esta escolha será respeitada para CADA PDF na pasta.
  4. Clique em "Processar Pasta...".
  5. Selecione a pasta de entrada (onde estão seus PDFs) e depois a pasta de saída (onde os resultados serão salvos).
Atenção: Se escolher "separadamente", o programa criará subpastas para cada PDF processado, mantendo tudo organizado.


Dicas e Boas Práticas

  • Use filtros curtos e genéricos. "Contestação" é melhor que "Peça de Contestação do Reclamado".
  • Para PDFs sem marcadores, o botão "Ver Todos os Marcadores" não mostrará nada. A extração só funciona em PDFs com marcadores.
  • Durante operações longas (como processar uma pasta grande), use o botão "Cancelar" para interromper a tarefa com segurança.
  • Organize seus filtros salvos (.json) em uma pasta para fácil acesso.


Solução de Problemas

Isso significa que o arquivo PDF que você abriu não possui marcadores (bookmarks). O Pinça Peças depende desses marcadores para identificar e extrair os documentos. Infelizmente, não há como processar este arquivo específico com o programa.

A barra de progresso deve indicar que o programa está trabalhando. Se ela não se mover por muito tempo, você pode usar o botão "Cancelar" para interromper a tarefa com segurança. Processar pastas com centenas de arquivos muito grandes pode levar algum tempo.
