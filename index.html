Bem-vindo ao Pinça Peças v2.0!

Sua suíte completa de automação para PDFs de processos, agora com uma interface renovada, mais rápida e intuitiva.

Este manual detalha todas as funcionalidades, incluindo o poderoso processamento em lote e a nova lista de seleção de documentos.

Baixar Pinça Peças


Instalação e Primeiro Uso

Siga estes passos simples para começar a usar o Pinça Peças.

  1. 1. Baixe o Programa
    Clique no botão de download acima para baixar o arquivo Pinca_Pecas.zip.
  2. 2. Descompacte o Arquivo
    Localize o arquivo .zip em sua pasta de downloads. Clique com o botão direito sobre ele e escolha a opção "Extrair Tudo..." ou "Descompactar aqui".
  3. 3. Execute o Pinça Peças
    Abra a pasta que foi criada. Dentro dela, dê um duplo clique no arquivo Pinca_Pecas.exe para iniciar o programa.
Aviso na Primeira Execução

Como o Pinça Peças é um programa novo, o Windows pode exibir uma tela azul de proteção (SmartScreen) ao executá-lo pela primeira vez. Isso é normal.

Para prosseguir, clique em "Mais informações" e, em seguida, no botão "Executar assim mesmo". O programa é seguro e isso só precisa ser feito uma vez.

Dica de Ouro!

Para facilitar o acesso, crie um atalho para o Pinca_Pecas.exe em sua Área de Trabalho.


As Ferramentas do Pinça Peças

O programa é organizado em três abas, cada uma sendo uma ferramenta poderosa e independente.

Extração por Peças

Para extrair documentos específicos (Petição, Sentença) de PDFs com marcadores.

Divisão por Páginas

Para quebrar qualquer PDF longo em partes menores com um número fixo de páginas.

Extração por Atos Recentes

Para "cortar" o início de um processo, mantendo apenas as movimentações mais recentes.


Funcionalidades em Detalhe

Aba: Extração por Peças
Quando usar? Para extrair documentos específicos de um processo com marcadores.

A função principal para organizar processos bem estruturados. Você define quais peças quer encontrar (ex: "Sentença", "Contestação") e o programa as extrai para você.

Como Usar a Nova Lista de Documentos:
  • Marcar/Desmarcar: Clique Simples no item ou selecione-o e pressione a Barra de Espaço.
  • Seleção Múltipla: Use Ctrl+Clique para selecionar itens individuais ou Shift+Clique para selecionar um intervalo, como em qualquer gerenciador de arquivos.
  • Pré-visualizar: Duplo Clique em qualquer item para abrir uma janela de preview do documento.
  • Lógica de Extração: O programa prioriza os itens marcados com o . Se nenhum item estiver marcado, ele extrairá os itens que estiverem selecionados (com fundo azul).
Busca por Conteúdo (Marcação Avançada)

Quando usar? Quando você precisa selecionar todos os documentos a partir de um evento específico que está descrito no texto do PDF, e não apenas no título do marcador.

  1. No campo "Marcar a partir do conteúdo:", digite uma frase ou termo único que identifica o início do período que você quer extrair. Exemplo: Homologo os cálculos.
  2. Clique em "Buscar e Aplicar". O Pinça Peças lerá o PDF página por página até encontrar o primeiro documento que contém esse texto.
  3. Automaticamente, ele marcará esse documento e todos os seguintes, desmarcando os anteriores.
Nota: A busca não diferencia maiúsculas de minúsculas. Uma janela de progresso será exibida durante a análise.
Recursos Adicionais:
  • Filtros Inteligentes: Adicione termos de busca que funcionam com lógica "OU". Salve e carregue seus conjuntos de filtros para reutilizá-los.
  • Pré-visualização Integrada: Dê um duplo-clique em qualquer documento da lista para visualizar suas páginas sem sair do programa.
  • Extração de Texto: Extraia todo o texto dos documentos selecionados para um arquivo .txt.
  • Estimativa de Tamanho: Veja o tamanho aproximado do arquivo PDF final antes mesmo de salvá-lo.
  • Opções de Salvamento: Junte todas as peças encontradas em um único PDF ou salve cada uma em um arquivo separado.
  • Processamento em Lote: Use o botão "Processar Pasta..." para aplicar seus filtros e regras a uma pasta inteira de processos de uma só vez.
Aba: Divisão por Páginas
Quando usar? Para qualquer PDF muito longo, especialmente aqueles sem marcadores.

A solução universal para tornar arquivos gigantescos mais gerenciáveis. Funciona com qualquer tipo de PDF.

Recursos Principais:
  • Divisão Simples: Abra um PDF, defina de quantas em quantas páginas quer dividir (ex: 100) e salve os arquivos resultantes.
  • Processamento em Lote: Use o botão "Dividir Pasta..." para aplicar a mesma regra de divisão (ex: "dividir a cada 100 páginas") para todos os PDFs dentro de uma pasta selecionada.
Aba: Extração por Atos Recentes
Quando usar? Para focar na parte final de um processo, ignorando o início.

Esta função especializada é ideal para preparar um processo para ser submetido a uma Inteligência Artificial que precisa avaliar apenas os últimos movimentos. Ela "corta" o início do PDF, criando um novo arquivo que contém apenas os acontecimentos mais recentes.

Recursos Principais:
  • Extração Focada: Abra um PDF de processo, escolha quantos dos últimos atos deseja manter (ex: 3, 5, 10) e o programa gera um novo PDF enxuto.
  • Processamento em Lote: Use o botão "Processar Pasta (Atos)..." para aplicar a mesma regra de extração (ex: "extrair os últimos 5 atos") para todos os PDFs de processo em uma pasta.
Nota: "Atos" são identificados por marcadores como "Despacho", "Decisão", "Sentença", "Acórdão", etc.


Dicas e Solução de Problemas

Isso acontece se os arquivos checked.png e unchecked.png não estiverem na mesma pasta do Pinca_Pecas.exe. Não se preocupe! O programa detecta isso e automaticamente usa os símbolos de texto ☐ e ☑. A funcionalidade permanece exatamente a mesma.

Isso quase sempre significa que o PDF não possui marcadores (bookmarks). Sem eles, as abas de extração inteligente não conseguem identificar os documentos. Solução: Utilize a aba "Divisão por Páginas" para quebrar o arquivo em partes menores e manejáveis.

Para operações demoradas, como a "Busca por Conteúdo" ou processamento em lote, uma janela de progresso será exibida. Se ela não se mover por muito tempo, você pode usar o botão "Cancelar" para interromper a tarefa com segurança. O processamento de muitos arquivos grandes pode levar um tempo considerável.
